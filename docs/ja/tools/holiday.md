# 🎌 holiday <Badge type="tip" text="Shell Script" /> <Badge type="tip" text="macOS" /> <Badge type="info" text="Linux" />

Google カレンダーの ICS フィードから日本の祝日を取得して一覧表示します。

## Requirements

- `curl`

## Usage

```bash
holiday [keyword]
```

| 引数 | Description |
|---|---|
| (なし) | 全祝日を日付順に表示 |
| `keyword` | キーワードでフィルタ |

# 今年の祝日を一覧表示
holiday

- ネットワーク接続が必要です。
