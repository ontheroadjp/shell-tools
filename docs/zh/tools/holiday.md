# holiday

从 Google 日历的 ICS 数据源获取日本法定节假日，并按日期顺序显示。

## 环境要求

- `curl`

## 用法

```bash
holiday [关键词]
```

| 参数 | 说明 |
|---|---|
| （无）| 按日期显示全部节假日 |
| `关键词` | 按关键词过滤节假日 |

## 示例

```bash
holiday
holiday 2025
holiday 天皇
```

## 输出示例

```
20250101 元日
20250113 成人の日
20250211 建国記念の日
...
```

## 说明

- 需要网络连接。
- 数据来源于 Google 日历的日本节假日日历（ICS 格式）。
