# 📜 wareki <Badge type="tip" text="Shell Script" /> <Badge type="tip" text="macOS" /> <Badge type="info" text="Linux" />

西历与日本年号互相换算，支持明治元年（1868年）至令和10年（2028年）。

## 用法

```bash
wareki [年份]
```

| 参数 | 说明 |
|---|---|
| （无）| 显示当前年份对应的日本年号 |
| `年份` | 按西历年份或年号名称搜索 |

## 示例

```bash
# 显示当前年份对应的日本年号
wareki
```

```
令和8年,2026
```

```bash
# 转换指定西历年份
wareki 2019
```

```
平成31年,2019
令和元年,2019
```

```bash
# 按年号名称搜索
wareki 昭和
```

```
昭和元年,1926
昭和2年,1927
...
昭和64年,1989
```

```bash
# 按部分年份搜索
wareki 1989
```

```
昭和64年,1989
平成元年,1989
```
