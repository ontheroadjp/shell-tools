# 📖 dictionary <Badge type="info" text="Python Script" /> <Badge type="warning" text="macOS" />

通过 macOS 内置 DictionaryServices 框架查询单词，支持英日互译并格式化输出。

## 环境要求

- Python 3
- macOS

## 用法

```bash
dictionary <单词>
```

## 示例

### 查询英文单词

```bash
dictionary serendipity
```

**Output:**

```
serendipity
  名詞  思いがけない幸運を見つける才能

▸ the faculty or phenomenon of finding valuable or agreeble things not sought for
```

### 查询日语单词

```bash
dictionary 曖昧
```

**Output:**

```
曖昧

ambiguous, vague, unclear
▸ 曖昧な答え  an ambiguous answer
```
