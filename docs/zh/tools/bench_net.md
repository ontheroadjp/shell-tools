# 🌐 bench_net <Badge type="tip" text="Shell Script" /> <Badge type="tip" text="macOS" /> <Badge type="info" text="Linux" />

通过 speedtest.net 测量网络下载和上传速度。

## 依赖

- Python 2 或 3

## 用法

```bash
bench_net
```

## 示例

```bash
# 通过 speedtest.net 测量网速
bench_net
```

```
Download: 487.32 Mbps
Upload:   198.54 Mbps
```

## 说明

- 内部使用 speedtest-cli 协议。
- 结果会因服务器负载和网络状况而有所不同。
